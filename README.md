# Smart Task Manager

## Description
A Smart Task Manager web app that allows users to create, manage, and track tasks efficiently with AI suggestions and voice commands.

## Features
- User login system  
- Add / delete / complete tasks  
- Filter tasks (All / Completed / Pending)  
- AI-based task suggestions  
- Voice command support  
- Dark mode  

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- Firebase  
- Netlify
